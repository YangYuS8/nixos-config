# NixOS flake for laptop (Ryzen 5 6600H / Radeon 680M)

Contains a NixOS flake to install a laptop-focused NixOS system:
- Btrfs with subvolumes: / (root), /home, /nix, /var
- Niri (Wayland compositor) + Noctalia (shell)
- fcitx5 + Rime + Chinese addons
- power management (TLP), PipeWire, zram

**Usage**
1. Boot NixOS live, partition & create btrfs subvolumes, mount them under `/mnt`.
2. Copy `hardware-configuration.nix` (generated by `nixos-generate-config`) into this repo.
3. Run `nix flake update` (optional), commit `flake.lock`.
4. From live environment: `sudo nixos-install --flake /path/to/repo#laptop`.

**Important**
- Do NOT store plaintext passwords in this repo. Use SSH keys or set password post-install.
- Tweak `flake.nix` placeholders (hostname, username, authorizedKeys) before install.
